# Booking
## How to start the program
+ prep
    - Download Flask
+ Run in terminal
    - set FLASK_APP=[students.py or admin.py]
    - FLASK run
+ open in web browser
    - localhost:5000

## Student
+ Signin
    - input code tex:AB12
+ Signup
    - Input
        - Firstname
        - Lastname
        - Activity
    - Confirmation

## Admin
